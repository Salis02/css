Display

Tag 'div' dan 'span' tidak memiliki 'arti' apapun, keduanya digunakan untuk mengelompokan tag-tag HTML dan memberikan informasi terhadap tag-tag tersebut. div memiliki properti display block dan span properti displaynya inline.

Apa hubungannya dengan display?

Setiap tag dalam HTML berada di dalam sebuah kotak. Properti dalam CSS mengatur perilaku dari kotak tersebut.Setiap tag pada HTML memiliki nilai default untuk properti display, tetapi kita juga dapat mengubah perilaku dari tag tersebut dengan mengganti value-nya.

Adapun nilai dari poperti display adalah:

  1. Inline
    • Elemen HTML yang secara default tidak menambahkan baris baru ketika dibuat
    • Lebar dan tingginya elemen sebesar konten yang ada di dalamnya, sehingga tinggi dan lebarnya tidak dapat diatur dari elemen inline
    • Margin dan padding hanya mempengaruhi elemen secara horizontal, tidak vertikal

      • Elemen inline
      b, strong, i, em, a, span, sub, sup, button, input, label, select, textarea,...

  2. Inline-block
    • Tidak ada elemen yang secara default memiliki properti display : Inline-block;
    • Harus diubah secara manual
    • Perilaku dasarnya sama seperti elemen inline
    • Bedanya elemen Inline-block dapat kita atur tinggi dan lebarnya

  3. Block

    Elemen block
    'h1-h6', 'p', 'ol', 'ul', 'li' , 'form', 'hr' , 'div' ....

    • Elemen HTML yang secara default menambahkan baris baru ketika dibuat
    • Jika tidak diatur width nya, maka lebar default dari elemen block akan memenuhi lebar browser/parentnya
    • Kita bisa mengatur tinggi dan lebar dari elemen block
    • Di dalam elemen inline blockk, kita dapat menyimpan tag dengan elemen inline, inline-block, atau bahkan elemen block lagi
