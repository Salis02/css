Pseudo Class

Pseudo Class merupakan kelas semu yang dimiliki oleh sebuah element HTML yang membuat kita dapat mendefiniskan style pada 'Keadaan Tertentu' dari elemen tersebut."

Pseudo-class yang berhubungan dengan link

  1. :link -> style default pada sebuah link (a yang ber-href)
    Pseudo-class dengan :link
  2. :hover -> style ketika kursor/mouse berada diatas sebuah link/elemen
    Pseudo-class dengan :hover
  3. :active -> style ketika sebuah link di-klik (keadaan aktif)
    Pseudo-class dengan :active
  4. :visited -> style ketika sebuah link sudah pernah dikunjungi sebelumnya (menggunakan browser yang sama)
    Pseudo-class dengan :visited

Pseudo-class yang berhubungan dengan posisi elemen

  1. :first-child
    -> memilih elemen pertama dari sebuah parent (elemen pembungkusnya)

    Contoh :

    Mengganti warna link elemen pertama

    • Link 1
    • Link 2
    • Link 3
    • Link 4
    • Link 5
    • Link 6
    • Link 7
    • Link 8
    • Link 9
    • Link 10

  2. :last-child
    -> memilih elemen terakhir dari sebuah parent (elemen pembungkusnya)

    Contoh :

    Mengganti warna link elemen terakhir

      Elemen 1

      Elemen 2

      Elemen 3

      Elemen 4

      Elemen 5

      Elemen 6

      Elemen 7

      Elemen 8

      Elemen 9

      Elemen 10

  3. :nth-child(n)
    -> memilih elemen ke-n dari sebuah parent (elemen pembungkusnya). n bisa berarti urutan (1),(2).... atau pola (2n), (3n+2), (4n-1) atau ganjil (even) dan genap (odd)

    Contoh :

    Mengganti warna link elemen genap

    • Elemen 1 Elemen 2 Elemen 3 Elemen 4 Elemen 5 Elemen 6 Elemen 7 Elemen 8 Elemen 9 Elemen 10
  4. :first-of-type
    -> memilih elemen pertama dari sebuah tag

  5. :last-of-type
    -> memilih elemen terakhir dari sebuah tag
