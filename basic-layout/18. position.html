CSS Position

Dalam materi kita sebelumnya, terutama margin, ketika kita akan memindahkan sebuah elemen ke bawah/atas, elemen dibawahnya akan ikut berpindah. Karena masalah itulah kita memerlukan sebuah properti CSS yang namanya Position, dimana kita bisa memindahakan elemen tanpa mempengaruhi elemen disekitarnya

Ada 4 value yang bisa kita gunakan dalam properti Position

    Static (nilai default)

    static adalah nilai default dari setiap elemen yang tidak diberi properti position.
    Menggunakan position selain static (non-static), akan membuat sebuah elemen seolah berbeda dimensi dari elemen lainnya.
    Elemen yang diberi position selain static dapat menggunakan properti top, left, bottom, right untuk mengatur posisinya

    Relative

    Ketika kita menggunakan elemen dengan posisi relative (menggunakan properti top, left, right, bottom), ruang yang ditempati oleh elemen tersebut masih ada, maju satu dimensi
    Ketika menggerakkan elemen dengan posisi relative, elemen akan bergerak relatif terhadap posisi semulanya
    Jika diberi properti top :0; left :0; maka elemen tidak berubah dari posisinya. Berikut penggunaannya :

    Absolute

    Ketika kita menggerakkan elemen dengan posisi absolute (menggunakan) properti top, left, bottom, right, ruang yang sebelumnya ditempati oleh elemen tersebut dianggap tidak ada

    Ketika kita menggerakkan elemen dengan posisi absolute, elemen akan bergerak relatif terhadap posisi dari elemen parentnya, selama elemen parentnya memiliki posisi yang non-static juga

    Jika kita memberi properti top : 0 ; left : 0 ;, maka elemen akan berada di ujung kiri atas elemen parentnya

    Fixed

    Perilaku dari position fixed, ketika kita memberikan properti top, left, bottom, right maka ia akan selalu bergerak relative terhadap elemen windownya

    Ketika kita scroll, elemen yang diberi position fixed, elemen tersebut akan selalu berada di posisinya. Hal ini sering digunakan oleh para web developer untuk membuat navigasi, agar selalu tampil ketika user melakukan scroll di website
