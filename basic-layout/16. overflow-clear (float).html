Float

Float merupakan properti pada CSS untuk mengatur posisi sebuah elemen, selain margin dan padding Sebuah elemen dapat dipaksa berada di sebelah kiri atau kanan dari parent/pembungkus-nya dengan menggunakan properti ini. Untuk properti Float memiliki beberapa value, yaitu

  • None (nilai default)
  • Left
  • Right

Perlu diketahui, bahwasanya di proses pembelajaran sebelumnya, kita telah menggunakan sesuatu (perilaku elemen ketika diberi CSS) yang disebut normal-flow. Dan untuk float ia termasuk dengan sesuatu yang dinamakan out-of-flow
yang bisa dilakukan menggunakan properti float adalah :

  • text wrapping
    membuat teks mengelilingi gambar/elemen lain

    • Ini adalah contoh text-wrapping dengan float dari img = right/left
