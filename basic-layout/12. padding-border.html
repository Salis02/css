Padding

Merupakan area transparan di dalam kotak (content dengan border). Aturan penulisan value sama dengan margin termasuk CSS Shorthandnya
Perbedaannya :

  • Padding tidak bisa diberi nilai negatif
  • Padding tidak bisa diberi nilai auto
  • Padding menambahkan ukuran elemen

Di dalam kotak ini diberi nilai paddingnya 10px (atas/bawah kanan/kiri = 10px)
Kita set height dan widthnya 300px, namun ketika kita tulis value padding maka akan bertambah nilainya.

Border

Batas keliling content dan padding. Border juga menambah ukuran elemennya
Penulisannya :
border : width style color

Ini adalah kotak yang diberi value
border : 7px solid yellow
Ini adalah kotak yang diberi value
border : 7px dotted yellow
Ini adalah kotak yang diberi value
border : 7px dashed yellow
Ini adalah kotak yang diberi value
border : 7px double yellow

Box Sizing

Biasanya untuk mengatur ukuran width dari sebuah box agar tidak berubah-ubah nilainya ketika ditambahkan padding/border
