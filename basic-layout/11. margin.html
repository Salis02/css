Margin

Overlapping Margin

  • Margin tidak saling menambahkan khusus untuk vertikal
  • Margin yang diambil secara vertikal adalah margin yang paling besar ukurannya, yang lebih kecil akan ditimpa
    • Big Box
    Diberi margin-bottom = 70px
    Medium Box
    Diberi margin-top 50px
    Small Box

    Margin medium box ditimpa oleh margin big box karena nilainya lebih kecil

  • Sedangkan secara horizontal, akan dijumlahkan kedua-duanya
    • Kotak satu
    Diberi margin-right 30px
    Kotak dua diberi margin-left 10px
    Kotak tiga

    Total jaraknya = kotak 1 (30px) + kotak 2 (10px) = 50px

Negatif Margin

Negatif margin akan membuat kotaknya bergerak berlawanan arah, ketika value setelahnya dideklarasikan. Biasanya digunakan untuk menyembunyikan suatu konten/elemen

Minus
Min
Min diberi sebuah nilai margin-top = -70px
Kemudian diberikan margin-left = 40px
Seharusnya Min akan geser kekiri, namun karena diberi nilai margin-top negatif maka gesernya berlawanan arah
Minuse
Perhatikan dengan seksama bahwa minuse ikut naik hanya ketika nilai margin Min diberikan secara vertikal, namun ketika horizontal ia tetap pada tempatnya

Auto

Auto akan membuat elemen ditengah-tengah parentnya, dan ingat kita harus menggunakan kedua margin-left dan margin-left nya dengan nilai auto yang sama

Kotak margin auto
Nilai yang diberikan :
margin-right: auto; margin-left: auto;
Kotak margin auto1
Nilai yang diberikan :
margin: 0px auto 0px
Kotak margin auto2
Nilai yang diberikan :
margin: 0px auto 0px

Shorthand

Dalam CSS Margin Shorthand, kita bisa mengisi value nya dengan :

  1. Satu nilai
    2. Shorthand 1 nilai
    Diberikan value :
    margin: 50px (atas/bawah/kanan/kiri = 50px)
  2. Dua nilai
    3. Shorthand 2 nilai
    Diberikan value :
    margin: 40px auto (atas/bawah = 40px; kiri/kanan : auto)
  3. Tiga nilai
    4. Shorthand 3 nilai
    Diberikan value :
    margin: 30px auto 30px; (atas = 30px; kanan/kiri = auto ; bawah = 30px)
  4. Empat nilai
    5. Shorthand 4 nilai
    Diberikan value :
    margin: 10px 20px 30px 40px (searah dengan jarum jam = atas -> kanan -> bawah -> kiri)
