Flex Box

"Model layout 1 dimensi yang dapat mengatur jarak dan penjajaran antar item dalam sebuah container."

1 dimensi?

"Hanya dapat mengatur 1 dimensi pada saat tertentu, antara baris atau kolom, tidak bisa keduanya sekaligus."

Istilah pada Flex-Box
  • main axis

    • => Sumbu utama dari sebuah container yang menentukan urutan dari penempatan items secara horizontal

  • main-start | main-end

    • => Mulai dan berakhirnya items yang disimpan di dalam container

  • main size

    • => Ukuran (width/height) dari container yang akan membuat dimensi dari itemsnya relatif terhadap main size

Dalam menggunakan properti flexbox, kita harus :

Displaynya kita ubah:
display : flex
Lalu kita kasih dibawahnya sebuah properti :
display : flex
flex-direction : row | row-reverse | column | column-reverse
Flex-direction => mengatur arah/urutan dari items di dalam container. NOTE : flex-direction memaksa elemen dalam satu baris!!!


Flex-wrap => secara default, semua items di dalam container akan berada pada satu baris meskipun ukurannya sudah tidak cukup, flex-wrap memungkinkan untuk memindahkan items ke baris dibawahnya.
Lalu kita kasih dibawahnya sebuah properti :
display : flex
flex-wrap : nowrap | wrap | wrap-reverse
Dengan flex-wrap, kita bisa memindahkan elemen ke baris selanjutnya!!!

Perbedaan

space-around space-between space-evenly
membuat elemen di start dan end memiliki jarak setengah dari jaraknya dengan elemen tengah dengan ujung container membuat elemen start dan end mepet dengan batas container, dan elemen diantaranya ditengah membuat jarak antara seluruh elemen sama rata kana kiri
